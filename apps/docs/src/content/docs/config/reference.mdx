## Top-level properties

#### `rootDir`

Root directory of your program

Default: `./src`

:::caution
The root directory cannot be the [`emitDir`](#emitdir)
:::

#### `include`

List of glob patterns to include in your program

Default: `['./**']`

```json
{
	"include": ["./src/**", "./tests/**"]
}
```

#### `exclude`

List of glob patterns to exclude when resolving [`include`](#include)

Default: `[]`

:::note
File extensions can be omitted since JS+ cannot parse files that don't end with `.jsp`
:::

:::caution
The `node_modules` and [`emitDir`](#emitdir) folders are excluded by default
:::

## `compiler`

Configuration for the compilation process

#### `emitLang`

Language to compile JS+ into. Can be `JavaScript` or `TypeScript`

Default: `TypeScript`

#### `emitDir`

directory to compile JS+ files into

Default: `./dist`

:::note
The emit directory cannot be the [`rootDir`](#rootdir) nor included in [`include`](#include).
:::

#### `emitSourceMaps`

Emit source maps in the [`emitDir`](#emitdir)

Default: `false`
