Scrap useless values when destructuring.

<Diff>
	```ts
	const [void, x, void, y, void, z] = [0, 1, 2, 3, 4, 5];

    const { z: void, ...obj1 } = { x: 1, y: 2, z: 3 };
    console.log(obj1); // { x: 1, y: 2 }
    ```

    ```ts
    const [_, x, __, y, ___, z] = [0, 1, 2, 3, 4, 5];

    let obj2;
    ({ z: _, ...obj2 } = { x: 1, y: 2, z: 3 });
    console.log(obj2); // { x: 1, y: 2 }
    ```

</Diff>

## Integration

This syntax is integrated with Babel's [plugin-proposal-discard-binding](https://babeljs.io/docs/babel-plugin-proposal-discard-binding) plugin.
