Due to legacy compatibility `typeof null` evaluates to `'object'`. [MDN docs](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Operators/typeof#typeof_null). [ECMAScript rejected proposal](https://web.archive.org/web/20160331031419/http://wiki.ecmascript.org:80/doku.php?id=harmony:typeof_null).

<Support support={frontmatter.support} />

:::caution
This syntax affects JavaScript original behaviour.
:::

<Diff>
	```ts
	typeof null; // 'null'
	```

    ```ts
    typeof null; // 'object'
    ```

</Diff>

## Integration

This syntax is integrated with a custom Babel plugin.
