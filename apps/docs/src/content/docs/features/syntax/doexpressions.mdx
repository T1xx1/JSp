Assign values based on a condition with better readability than ternaries.

<Support support={frontmatter.support} />

<Diff>
	```ts
	const x = 100;
	const y = 20;

    const a = do {
    	if (x > 10) {
    		if (y > 20) {
    			'big x, big y';
    		} else {
    			'big x, small y';
    		}
    	} else {
    		if (y > 10) {
    			'small x, big y';
    		} else {
    			'small x, small y';
    		}
    	}
    };
    ```

    ```ts
    const x = 100;
    const y = 20;

    const x = x > 10
    		? y > 20
    			? 'big x, big y'
    			: 'big x, small y'
    		: y > 10
    			? 'small x, big y'
    			: 'small x, small y';
    ```

</Diff>

:::caution
Do expressions are different than [Async do expressions](/syntax/asyncdoexpressions)
:::

## Integration

This syntax is integrated with Babel's [plugin-proposal-do-expressions](https://babeljs.io/docs/babel-plugin-proposal-do-expressions) plugin.
