Async do expressions can replace async IIFEs called in a synchronous scope.

<Support support={frontmatter.support} />

<Diff>
	```ts
	import { readFile } from 'node:fs';

    async do {
    	const file = await readFile('file.txt', 'utf8');

    	console.log(file);
    }
    ```

    ```ts
    import { readFile } from 'node:fs';

    (async () => {
    	const file = await readFile('file.txt', 'utf8');

    	console.log(file);
    })();
    ```

</Diff>

:::caution
Async do expressions are different from [Do expressions](/syntax/doexpressions).
:::

## Integration

This syntax is integrated with Babel's [plugin-proposal-async-do-expressions](https://babeljs.io/docs/babel-plugin-proposal-async-do-expressions) plugin.
